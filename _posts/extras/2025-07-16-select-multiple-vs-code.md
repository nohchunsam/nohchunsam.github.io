로그를 Vs Code로 여는데... 필요없는 로그가 한 1000000개 이상 됐다. 이걸 한꺼번에 삭제하는 방법이 없을까 계속 찾다가 겨우 발견해서 기록해둔다.

1. 원하는 줄~선택 영역의 첫번째 줄을 클릭/선택
2. `Ctrl+G`로 원하는 줄 넘버 타이핑(만!!) -> 자동 이동
3. `Shift 누른채로 이동해서 도착한 줄 클릭
4. profit
